Presentación
--------
ZEETY MD es un marco de desarrollo de aplicaciones orientadas
a la web en servidores Apache (recomendado sobre sistemas
operativos GNU/Linux), teniendo como base de la capa de 
persistencia el gestor de bases de datos MySQL o cualquier
otro motor libre como postgreSQL, MariaDB, Firebird (Por
cuestiones de compatibilidad Zeety MD también incluye 
soporte para ORACLE y a otros motores no libres via ODBC)

Zeety MD es un estilo arquitectónico modular con elementos
y herramientas predefinidas que aceleran el desarrollo de
aplicaciones. 

Su principal aplicación es en el proyecto ZEETY (Sistema de
Inteligencia Territorial UD), pero su grado de estabilidad
ha permitido que se utilice en otros proyectos.

Disponibilidad
------------

La última versión de desarrollo de Zeety MD está disponible 
en el repositorio público: https://github.com/zeety/Zeety_MD.git.

Instalación
------------
1. Descargar los archivos desde el repositorio de
github.

2. Colocarlos en una carpeta del servidor web.

3. Desde un navegador web ingresar a la carpeta.
